Site Description and Purpose

Client requested student developer build a responsive website of consistent design suitable for viewing on all classes of devices throughout. The site informs users of the current weather conditions, weather forecast, news and events in and around the towns of Preston, Soda Springs, and Fish Haven in Southeastern Idaho. The site includes the following pages: Home, Preston, Soda Springs, Fish Haven, Storm Center, and Gallery. Home page displays current weather discussion with corresponding weather image and information sections for the towns of Preston, Soda Springs, and Fish Haven. Town pages contain news, events, other community information and happenings in each town. Page footers contain contact information, location map, copyright, author's name, terms of use link, and current date.

The site layout is of an intuitive design based on common features and navigation styles for each device. The site is designed for quick access to information so no login account will be required to use the site.


Site Title

Weather in Southeastern Idaho (WISEID). A shortened version of the title could be WISEID to match the domain name.


Domain Name

Student developer proposes the domain name of wiseid.info which is available for $12 annually through Google Domains.


Site Logo

Site logo, appearing on each page, was obtained as a free logo with creator's attribution. This can easily be replaced by a custom logo of client's choosing. This owl icon fits with the moto of making a wise choice (see site moto below).
